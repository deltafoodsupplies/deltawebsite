# Delta Food Supplies Website

Static GitHub Pages site for Delta Food Supplies.

## Files

- `index.html` - page content and SEO metadata
- `styles.css` - responsive layout and visual design
- `script.js` - mobile navigation behavior
- `assets/` - Delta logo assets used by the site

## GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository, open `Settings` > `Pages`.
3. Set the source to deploy from the main branch root.
4. Save, then wait for GitHub Pages to publish the site.

The contact form opens email inquiries addressed to `sales@deltafoodsupplies.com`.
